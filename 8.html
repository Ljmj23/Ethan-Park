Hold something

Mix media

For my favorite material, I used Korean traditional paper that is translucent which light can also pass through. I was inspired by traditional Korean lamps which can be held, and I made a shield-like shape that I can hold. The original function of the lantern is to hold and light up the surroundings so we can walk on the street. However, I incorporated the opposite way and created an object with two roles that can block and hide myself from the surrounding. In order to show the function of blocking from the surrounding area, I added the different shapes using the Korean tradition
